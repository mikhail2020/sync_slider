Этот проект представляет собой слайдер, разработанный с использованием React, TypeScript, SASS, GSAP, Swiper.

## Описание
Основной компанент SyncSlider принимает следубщие свойства:

```typescript
interface SyncSliderProps {
    title?: string;
    data: DateItem[];
    timePeriods: 'two' | 'three' | 'four' | 'five' | 'six';
}

type DateItem = {
    year: string;
    event: string;
    theme?: string;
};

title - заголовок (Не обязательное свойство).
data - масив с данными для отображения (тип данный DateItem) 
timePeriods - разделяет даты, от 2 до 6 временных отрезков.

При совпадении тем у всех событий в одном временном отрезке, около выбранного отрезка отображаеться соответствующая тема.

## Требования 
Установленные: 
**Node.js** (рекомендуемая версия: 14.x или выше),
**npm** (или **yarn**) — менеджеры пакетов для установки зависимостей.
**Git**


## Установка
Склонируйте удаленный репозиторий, используйте следующую команду для копирования репозитория на ваш компьютер:
git clone https://github.com/mikhail2020/sync_slider.git

## Запуск
перейди в директорий с проектом
cd sync_slider

Установите все необходимые зависимости
npm install
или
yarn install

Запустите проект
npm start
или
yarn start

## Используемые технологии
React.js: Библиотека для построения пользовательских интерфейсов.
TypeScript: Язык программирования, который добавляет статическую типизацию к JavaScript.
SASS/SCSS: Препроцессор CSS, который упрощает стилизацию приложения.
GSAP: Библиотека для создания анимаций с высокой производительностью.
Swiper: Библиотека для создания адаптивных слайдеров.
